## Описание:

    Основные модули
        main.py - Главная точка входа в программу
        
        call_logic.py - Логика обработки команд и вызовов алгоритмов, включая команду benchmark
        
        benchmark.py - Функции для проведения бенчмаркинга алгоритмов сортировки и вывода результатов
        
        sorts_algorithms.py - Реализации различных алгоритмов сортировки:
        
            Bubble Sort
            
            Quick Sort
            
            Counting Sort
            
            Radix Sort
            
            Heap Sort
 
            Bucket Sort
        
        math_algorithms.py - Математические алгоритмы

        data_structures.py - Реализации структур данных (на list)



    Генерация тестовых данных
    tests_case_generator.py - Генератор тест-кейсов для создания разнообразных массивов используемых в тестах 
    и для генерации массивов для бенчмаркинга:
    
        rand_int_array() - случайные целочисленные массивы
        
        nearly_sorted() - почти отсортированные массивы
        
        many_duplicates() - массивы с большим количеством дубликатов
        
        reversed_sorted() - массивы в обратном порядке

        rand_float_array() - случайные вещественные массивы
    
    arr.py - Предварительные тестовые массивы для бенчмаркинга



    Бенчмаркинг
    Для проведения бенчмарка используется команда benchmark, которая:
    
        Тестирует все алгоритмы сортировки на идентичных наборах данных
        
        Использует генерацию тест-кейсов для создания разнообразных массивов
        
        Измеряет время выполнения каждого алгоритма
        
        Выводит сравнительные результаты

## Особенности работы:

    1. Все названия параметров следует вводить корректно через 1 пробел, иначе будет вызвана соответствующая ошибка
    2. Массив должен быть передан в квадртаных скобках через запятую -> пример: [12, 2321, 123]
    3. Параметр для bucket должен быть передан через знак равно -> пример: bucket=5
    4. Программу можно заврешить Ctrl+f2/Ctrl+C/Ctrl+D
    5. Для bucket sort значения только из промежутка [0, 1)

    В начале работы программы выводится список доступных комманд и как их следует вводить
